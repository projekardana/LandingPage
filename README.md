## 🎯 Tentang Proyek

**Pustaka Muslim Official Store Landing Page** adalah website modern untuk toko buku online yang menjual koleksi lengkap buku-buku Islamic Studies. Proyek ini dirancang dengan prinsip **Mobile-First Responsive Design** menggunakan teknologi **HTML5, CSS3, dan Vanilla JavaScript** tanpa dependency framework tambahan.

Website ini menampilkan:
- 📚 Koleksi produk buku Muslim berkualitas
- 💬 Testimoni pelanggan yang puas
- 📰 Artikel-artikel menarik
- ✉️ Newsletter subscription
- 🌙 Dark mode support
- 📱 Fully responsive design

### **JavaScript Features**

#### **1. Dark Mode Management**
```javascript
// Toggle dark mode dengan localStorage persistence
darkModeToggle.addEventListener('click', () => {
  body.classList.toggle('dark-mode');
  localStorage.setItem('darkMode', body.classList.contains('dark-mode'));
});
```

#### **2. Mobile Navigation**
```javascript
// Hamburger menu dengan auto-close
hamburger.addEventListener('click', () => {
  hamburger.classList.toggle('active');
  navList.classList.toggle('active');
});
```

#### **3. Scroll Animation**
```javascript
// Intersection Observer untuk scroll animation
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.style.opacity = '1';
    }
  });
});
```

#### **4. Testimonial Carousel**
```javascript
// Auto-rotating testimonial carousel
setInterval(() => {
  currentTestimonial = (currentTestimonial + 1) % testimonialCards.length;
  showTestimonial(currentTestimonial);
}, 5000);
```

#### **5. Newsletter Validation**
```javascript
// Email validation dengan regex
const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
if (!emailRegex.test(email)) {
  showMessage('Mohon masukkan email yang valid', 'error');
}
```

## 🎨 Customization Guide

### **Mengubah Brand Colors**
Edit CSS variables di `Style/app.css`:
```css
:root {
  --primary-color: #YOUR_COLOR;
  --secondary-color: #YOUR_COLOR;
  /* ... */
}
```

### **Menambah Testimoni Baru**
Edit HTML di `index.html`:
```html
<div class="testimonial-card">
  <div class="stars">
    <!-- Rating stars -->
  </div>
  <p class="testimonial-text">Your testimonial here</p>
  <p class="testimonial-author">- Name</p>
</div>
```

### **Menambah Produk Baru**
```html
<div class="product-card">
  <div class="product-image">
    <img src="img/product.jpg" alt="Product">
    <span class="badge">Badge Text</span>
  </div>
  <h3>Product Name</h3>
  <p class="price">Rp XXX.XXX</p>
  <p class="description">Product description</p>
  <button class="btn-add-cart">Add to Cart</button>
</div>
```

### **Mengubah Fonts**
Edit Google Fonts link di `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT:wght@300;400;600;700&display=swap" rel="stylesheet">
```
